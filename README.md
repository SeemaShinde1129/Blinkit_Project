# Blinkit Sales Analysis Project

## Overview
This project focuses on analyzing Blinkit's sales data to understand business performance, customer behavior, and product trends.  
I used SQL, Excel, Power BI, and Python to clean, process, and visualize the data.

---

## Dataset Description

The dataset contains information about products, sales, and outlets. Below are the main columns used:

- **Item Fat Content** → Type of product (Low Fat / Regular)
- **Item Identifier** → Unique ID for each product
- **Item Type** → Category of product (Dairy, Snacks, Fruits, etc.)
- **Outlet Establishment Year** → Year when the outlet was started
- **Outlet Identifier** → Unique ID for each store
- **Outlet Location Type** → Location tier (Tier 1, Tier 2, Tier 3)
- **Outlet Size** → Size of the outlet (Small, Medium, High)
- **Outlet Type** → Type of store (Grocery Store, Supermarket)
- **Item Visibility** → How visible the product is in store
- **Item Weight** → Weight of the product
- **Sales** → Total sales amount
- **Rating** → Customer rating of the product

---

## Project Objective
The main goal of this project is to:
- Analyze total sales and average sales
- Understand customer preferences using ratings
- Identify top-performing products and outlets
- Study how outlet size, type, and location affect sales

---

## Key Analysis Performed

- Total Sales Analysis
- Average Sales per Product
- Number of Items Sold
- Average Customer Rating

### Charts Created:
- Sales by Item Fat Content (Donut Chart)
- Sales by Item Type (Bar Chart)
- Sales by Outlet (Stacked Column Chart)
- Sales by Outlet Establishment Year (Line Chart)
- Sales by Outlet Size (Pie/Donut Chart)
- Sales by Outlet Location (Map/Funnel)
